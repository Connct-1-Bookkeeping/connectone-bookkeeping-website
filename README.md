## Publish free with GitHub + Netlify
1. Create a new GitHub repository.
2. Upload ALL files from this folder to the repository root.
3. In Netlify, choose "Add new project" > "Import an existing project".
4. Connect GitHub and select the repository.
5. Build command: leave empty.
6. Publish directory: .
7. Deploy.
8. In Netlify, open Domain management and add connectonebookkeeping.com.au.
9. Follow Netlify's DNS instructions for your Crazy Domains DNS records.

## Contact form
The contact form uses Netlify Forms. After the site deploys through Netlify, form submissions appear in the Netlify dashboard under Forms.
